Scrapy制作搜索引擎(一)

环境搭建

1.pycharm的安装 下载专业版（http://www.jetbrains.com/pycharm/）
2.安装mysql和navicat
3.安装python3（windows下需配置python路径以及pip路径）
4.安装虚拟环境
安装virtualenv
pip install virtualenv
安装virtualenvwrapper
pip install virtualenvwrapper-win
修改虚拟环境地址
windows环境变量中加入WORKON_HOME 为自己创建的Envs文件夹地址
新建虚拟环境
mkvirtualenv --python="需要的python版本路径" envName(环境名) (若使用自带版本，则不需要加python参数)
查看虚拟环境
workon
进入虚拟环境
workon envName(环境名)
退出虚拟环境
deactivate

使用虚拟环境创建项目

打开pycharm 点击File->new Project->Pure Python
选择 Existing interpreter 选择Envs地址下的需要使用的虚拟环境envName下的Scripts下的python.exe
修改最上方location(项目地址)为创建项目的地址,点击创建（create）即可
