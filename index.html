Scrapy制作搜索引擎（一）

环境搭建

  1. pycharm 的安装 下载专业版（http://www.jetbrains.com/pycharm/）
  2. 安装 mysql 和 navicat
  3. 安装 python3（windows 下需配置 python 路径以及 pip 路径）
  4. 安装虚拟环境
    安装 virtualenv
    pip install virtualenv
    安装 virtualenvwrapper
    pip install virtualenvwrapper-win
    修改虚拟环境地址
    windows 环境变量中加入 WORKON_HOME 为自己创建的 Envs 文件夹地址
    新建虚拟环境
    mkvirtualenv --python="需要的python版本路径" envName(环境名) (若使用自带版本，则不需要加 python 参数)
    查看虚拟环境
    workon
    进入虚拟环境
    workon envName(环境名)
    退出虚拟环境
    deactivate

使用虚拟环境创建项目

打开pycharm 点击File->new Project->Pure Python
选择 Existing interpreter 选择Envs地址下的需要使用的虚拟环境envName下的Scripts下的python.exe
修改最上方location(项目地址)为创建项目的地址,点击创建（create）即可
